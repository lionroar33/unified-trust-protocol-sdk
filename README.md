# unified-trust-protocol-sdk
Unified Trust Protocol (UTP) is an open-source "Trust Layer" for Solana PayFi. We stop Sybil attacks/fraud by combining ZK-Email validation (verifying aged domains privately) and Biometric Liveness via the Solana Mobile Stack.
